# 系統的作用

## 線上訂房系統

- 讓使用者透過此系統的簡單操作就可以預約房間
- 讓使用者隨時掌握房間的最新狀況
- 讓管理者在一套系統裡就可管控所有顧客的預約資料
- 讓管理者可以隨時修改房間的狀況
